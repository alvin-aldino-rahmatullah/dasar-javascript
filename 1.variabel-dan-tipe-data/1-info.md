variabel berfungsi sebagai wadah untuk menyimpan data atau nilai dalam program kita.
variabel memungkinkan kita untuk menyimpan, mengubah, dan mengakses data dengan mudah selama program berjalan. 

tipe data adalah kategori atau jenis data yang dapat disimpan dalam variabel.
tipe data yang umum digunakan dalam pemrograman antara lain
1. number (angka)
2. string (teks)
3. boolean (benar atau salah)
4. null (nilai kosong)
5. undefined (variabel yang belum memiliki nilai)

dengan menggunakan variabel dan tipe data, kita dapat membuat program yang dinamis dan interaktif, karena kita dapat menyimpan informasi, memanipulasi data, dan mengambil keputusan berdasarkan nilai yang disimpan dalam variabel tersebut. Singkatnya, variabel dan tipe data adalah konsep dasar yang sangat penting dalam pemrograman untuk mengelola dan memanipulasi data dalam aplikasi kita.
